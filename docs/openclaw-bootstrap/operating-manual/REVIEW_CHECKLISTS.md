## Using Checklists

### When Submitting to REVIEW

1. Copy the appropriate checklist template
2. Check each item honestly
3. Add notes explaining any unchecked items or special considerations
4. Include the checklist in your submission:

```bash
mc task submit task_123 \
  --deliverable ./output/deliverable.md \
  --checklist ./output/checklist.json
```

### As a Reviewer

When reviewing another agent's work:
1. Verify each checked item
2. Test claims where possible
3. If items are unchecked, assess if blocking
4. Provide feedback with specific checklist references

  type: TaskType;
